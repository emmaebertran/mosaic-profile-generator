# Mosaic profile photo generator

A static browser app that turns a pasted, dropped, or uploaded profile photo into a Mosaic-style triangular portrait using the Mosaic brand palette.

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.

## Deploy

This site is fully static. Deploy the folder root to any static host:

- Netlify
- Vercel
- GitHub Pages
- Cloudflare Pages

No build step is required.
